
1. What is JSX, and why is it used?
Ans: JSX means JavaScript XML is a syntax for JavaScript used in React that allows developers to write as like HTML code inside JavaScript.

2. What is the difference between State and Props?
Ans: State: State is data managed inside a component
Props: Props are read only data passed from a parent component to a child component.

3. What is the useState hook, and how does it work?
Ans: useState is a React Hook that allows functional components to store and manage data.

4. How can you share state between components in React?
Ans: Lifting State Up: When multiple components need the same data the state is moved to their nearest common parent component. 
Using Props: State can be passed from a parent components to child components using props.
Context API: Sharing state across many components
State Libraries: Managing complex global state

5. How is event handling done in React?
Ans: Event handling in React is used to respond to user action like clicks, typing, form data submit, mouse movements etc.
